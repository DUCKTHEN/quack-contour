# Release Notes

## 0.1.0 beta draft

- Local OBJ section viewer.
- Primary and comparison model import.
- Silhouette and outline display modes.
- Section guide visibility and color controls.
- Top-view PNG/JSON generation.
- Underlay image movement, scale, and rotation.
- Doodle tools for temporary notes.
- Rectangle and lasso mesh masking.
- Doodle color picker with lower-latency pen preview.
- Model diagnostics panel for dense meshes, unusual scale, center offset, and floor offset.
- Comparison model default opacity is now 100%, matching the primary model.
- Section anomaly warning marks in the measurement list.
- Local privacy-first repository layout.
- Coalesced viewer redraws with requestAnimationFrame for smoother drag, mask, doodle, and resize interactions.
- Capped interactive top-view section scanning to keep browser-triggered generation responsive.
