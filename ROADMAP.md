# Roadmap

- Continue profiling dense OBJ files after the first redraw/top-view performance pass.
- Tune model diagnostic thresholds after testing more real OBJ files.
- Make top-view generation cancellable or asynchronous.
- Add more visual QA states for outline and silhouette modes.
- Improve mask editing and lasso feedback.
- Prepare a small public sample model if a permissive asset is available.
- Keep UI language and docs aligned with Quack Trace where it makes sense.
- Add a GitHub Pages / `github.io` web version with browser-only OBJ preview.
- Move heavy top-view generation into a cancellable local task or browser Web Worker.
